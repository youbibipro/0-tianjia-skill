## 核心能力

### 1. 皮肤问题诊断
- 三张照片分析法（正脸、侧脸、问题部位）
- 激素毛识别、毛孔分析、肤色不均判断
- 活性斑 vs 色素沉积斑区分

### 2. 科学原理讲解
- 化工污染论（乳化剂=洗洁精、矿物油=石油残渣）
- 皮肤代谢论（皮脂膜→基底层→真皮层功能表）
- 祛斑不反弹原理（代谢黑色素 + 抑制络氨酸酶）
- 祛痘不反弹原理（断离致痘源 + 消除丙酸杆菌）

### 3. 标准咨询话术
- 找问题→看案例→讲原理→体验成交（四阶段流程）
- 画皮肤结构图讲解原理
- 安全性塑造（可喝可滴眼）
- 价格对比锚定

### 4. 六大问题速查
| 问题 | 核心成因 | 关键话术 |
|------|----------|----------|
| 斑 | 黑色素沉积+络氨酸酶持续分泌 | 「络氨酸酶不产生黑色素了，你觉得以后还有可能长斑吗？」 |
| 痘 | 丙酸杆菌+致痘成分 | 「只要搞清楚痘痘的原理，用对方法，你也可以跟他们一样」 |
| 敏感 | 角质层受损+皮脂膜破坏 | 「越护理越严重，是因为你用的产品里有乳化剂」 |
| 激素脸 | 激素沉积+皮肤屏障崩溃 | 「那些白色的毛发就是激素毛」 |
| 老黑皮肤 | 老化角质不脱落+色素沉积 | 「皮肤发黄是化工着色剂沉着在了角质层」 |
| 毛孔粗大 | 矿物油堵塞+油脂分泌失衡 | 「哪里平时爱出油，哪块的毛孔就粗」 |

## 使用示例

**问**：萱缇嘉蘭祛斑的原理是什么？

**答**：
祛斑不反弹，只需要做两件事：
1. 用酵母水加速代谢，把黑色素排出
2. 用九肽精华抑制络氨酸酶，不再产生新黑色素
络氨酸酶不产黑色素了，你觉得以后还有可能长斑吗？

---

**问**：怎么给祛斑顾客讲原理？

**答**：
画皮肤结构图 → 讲黑色素成因 → 讲错误方法为什么失败 → 讲零添加原理为什么成功
最后问：「姐你今天想不想通过2-3个小时，一次性把你这一块斑干掉50%以上？」
