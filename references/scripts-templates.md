# 话术模板库

## 一、找问题阶段

### 照片分析开场
```
"你好，我来帮你拍张照，看一下你的皮肤，看一下怎么针对性安排今天的护理流程。"
```

### 正脸分析（肤色）
```
"肤色稍微有点不均，你看这里的皮肤，就会比这里的颜色重一点，这里的皮肤就会比这边重一点。"
```

### 侧脸分析（毛孔）
```
"我们再看第二张照片，你会发现这一块毛孔可能有点粗，你有没有发现这样一个问题，哪里平时爱出油，哪块的毛孔就粗，所以说明这一块平时是不是特别爱出油？"
```

### 侧脸分析（激素毛）
```
"然后，这个部位还有一些白色的毛发，我不知道你有没有留意，这些叫做激素毛，说明你皮下有激素沉淀，之前用的一些护肤品，可能你不知道里边有激素成分，一般能快速、当下就让你觉得皮肤状态变得非常好，但殊不知的是，已经在皮下注入激素了。"
```

### 侧脸分析（皱纹）
```
"下来你会看到有一些幼纹、还有一些皱纹、小细纹，这说明什么呢？你之前用的一些产品或者是护肤的过程中，里边有一个成分叫乳化剂，乳化剂会导致你皮肤加速衰老，今天过会给您护理的时候，我让员工帮你在这一块多拍水，帮你去把这块的皱纹淡化。"
```

### 问题部位分析（斑）
```
"下来就是看到最明显的就是斑，你的斑多少年了？有没有生气或情绪波动的时候，斑就变重了？不生气的时候斑就浅了？"
```

### 询问历史
```
"用过什么方法祛过没？用过的话都是什么反应？"
```

## 三、讲原理阶段

### 吸引听原理
```
"姐，你想不想知道这到底是怎么做到的，我给你讲一下斑是怎么来的，以及真正正确的治斑原理行不？好多人都不知道这些真相，甚至还有很多无良商家为了卖点东西，特意制造很多错误观念，不但斑没去掉反弹了，还把皮肤搞得更糟糕了；当你听我讲完这些原理，你未来哪怕不找我帮你解决斑，也不至于被别人忽悠。"
```

### 画图讲解（斑成因）
```
"我画一条线，这代表我们的皮肤，皮肤有一个功能，就是吸收好东西进来，同时代谢垃圾出去，就是我们皮肤的代谢功能。你会发现18岁不长斑，28岁不长斑，但是随着年龄再长大，斑就开始越来越多了，从一点到一片，源于细胞死亡沉淀的黑色素，或者沉淀了护肤品中的色素成分，皮肤代谢一慢，它就开始堆积越来越多，最终形成一片了。"
```

### 讲错误方法（皮秒）
```
"第一种办法就是仪器皮秒，通过射频的原理，把这一块的斑用射频振碎，它本来是很大一个颗粒，给它振散、振散，你会发现斑就慢慢变淡了，所以达到祛斑的效果。但你会发现，这对皮肤的角质层造成了不可逆的伤害。"
```

### 讲正确方法
```
"那零添加能够通过哪几个步骤，做到不反弹祛斑呢？第一个，有一款水，这是西班牙的靶向酵母，加速面部代谢的，帮你把皮肤下面的深层和浅层的斑，通过加速代谢的原理，把它们全部代谢出来，能达到1个小时或者是5个小时，就能把你能看得到的斑，就淡化70—80%。第二个，就要解决反弹的问题，就是抑制络氨酸酶的产生，不让它每天都产生黑色素了，自然而然，斑就不会再有了。"
```

## 五、祛痘专项话术

### 祛痘原理开场
```
"你会不会发现一个问题，男人油脂分泌旺盛一点，脸上好像很少长痘啊，就算是偶尔长了，不用管它，到三四十岁的时候痘就自然好了，那这背后是啥原因呢？你看，很多人一直花钱祛痘，但你发现痘反而还越来越多了，而且毛孔还越来越大了，皮肤越来越糟糕了；这是因为很多人不了解痘的真正原理，被无良商家误导了。"
```

### 丙酸杆菌讲解
```
"丙酸杆菌有一个特性是"喜油厌氧"，俗称"厌氧菌"，你们有没有发现一个问题，皮肤油的人、痘很多，而且毛孔很大，但是皮肤干的人从来不长痘。因为"厌氧菌"就是喜欢油的地方，没有氧气的地方；你会发现有一些人，30岁之前反复长痘，一过30岁、40岁或者50岁，痘痘不治而愈了，原因是为什么呢？就是因为皮肤代谢开始变慢了，皮脂膜分泌的油脂开始变少了，那油脂一少，毛孔自然而然就通畅了，毛孔是人体第二大呼吸系统，它能去吸收氧气进到毛孔里头，丙酸杆菌就没有生存的环境了。"
```

### 祛痘方式选择
```
"咱们的祛痘方式有两种：一种是常规性的，就是稳定祛痘，就是由浅入深的逐步拔出"丙酸杆菌"；还有一种是更快速的，就是用酵母水中的二列靶向酵母，快速把皮肤深层的丙酸杆菌代谢出来；这种祛痘周期更短，但是因为把皮下的和深层的丙酸杆菌拔出来了，可能在短时间会痘痘更多了；您看您是选择常规型还是加速型的？"
```

### 祛痘承诺
```
"你看你这痘旁边有炎症有点红肿，可能还会有点痒对吧，一会护理40分钟到1小时的时候，炎症起码消除60%到70%。"
```
