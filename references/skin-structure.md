# 皮肤结构功能表

## 表皮层（由外到内）

### 皮脂膜
- **组成**：皮脂腺分泌的皮脂 + 汗腺分泌的汗液，在皮肤表面乳化形成
- **pH值**：4.5-6.5（弱酸性）
- **功能**：
  1. 润肤与锁水（保持皮肤柔软光滑）
  2. 抑制有害微生物生长
  3. 维持角质层正常代谢
- **受损后果**：干、敏感、代谢停滞

### 角质层
- **组成**：10-20层无核扁平角质细胞
- **功能**：
  1. 抓取空气中的水分（乳酸钠、乳酸钾）
  2. 锁住皮肤中的水分
  3. 保持弱酸性环境
  4. 防止摩擦刮蹭
- **受损后果**：薄、老化角质不脱落、粗糙

### 透明层
- **组成**：3-4层无核角质细胞
- **分布**：只在手掌和足底明显
- **功能**：增强屏障功能和耐磨性能

### 颗粒层
- **组成**：3-5层梭形（菱形）角朊细胞，内含透明角质颗粒
- **功能**：
  1. 折射紫外线（承受15分钟照射）
  2. 生成角质层（角质蛋白+角质小体+失去活性的角质细胞）
- **受损后果**：紫外线伤害直达基底层

### 有棘层
- **功能**：
  1. 启动免疫修复（朗格汉斯细胞识别外来抗原）
  2. 抵御细菌、病毒、有害物质（香精、防腐剂）
  3. 促进伤口修复和组织重建
  4. 养育新生角质细胞（给新生细胞提供营养）
- **受损后果**：易感染、修复慢、角质异常

### 基底层
- **组成**：
  - 基底母细胞（柱状/立方形，15-30天更新一次）
  - 黑色素细胞（决定皮肤颜色）
  - 络氨酸酶（受刺激后分泌黑色素）
- **功能**：
  1. 不断产生新细胞向外迁移
  2. 紫外线/激素刺激时分泌黑色素保护皮肤
- **受损后果**：色斑、皮肤更新慢

### 毛孔毛囊
- **功能**：
  1. 分泌排泄通道（皮脂、汗液排出）
  2. 辅助呼吸（气体交换）
  3. 感受外界刺激（触觉、温度、湿度）
- **受损后果**：堵塞、毛孔粗大、炎症

## 真皮层

### 胶原纤维
- **功能**：支撑皮肤，让皮肤不塌陷
- **受损后果**：松弛、皱纹

### 弹力纤维
- **功能**：抵抗地球重力
- **受损后果**：下垂

### 黏多糖
- **功能**：填充在胶原纤维和弹力纤维之间的空隙，提供体积和支撑力
- **受损后果**：空洞、凹陷

## 化工成分对照表

| 化工成分 | 产品中的作用 | 实际危害 |
|----------|-------------|----------|
| 乳化剂 | 水油相融形成乳/膏/霜 | =洗洁精原理，破坏皮脂膜，皮肤越来越薄 |
| 矿物油 | 让皮肤湿润不干燥 | =石油残渣，无法代谢，堵塞毛孔，纤维板结硬化 |
| 防腐剂 | 防止产品变质 | 杀死皮肤表面有益菌群，破坏弱酸性环境 |
| 色素着色剂 | 产品颜色美观 | 沉积在角质层，皮肤发黄发暗 |
| 香精 | 产品气味好闻 | 刺激有棘层，引发过敏反应 |

## 护肤品安全判断标准

**可喝可滴眼**：如果一款护肤产品可以喝、可以滴眼睛，说明它绝对安全零添加。

**成分查询工具**：用「你今天真好看」APP查询药监局备案成分，确认无化工成分。
